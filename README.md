# Neptune 小程序

## 项目简介
Neptune 是一个由学生为学生打造的资源流动平台。我们的目标是让闲置资源流动起来，没有大公司的参与，完全由我们学生自己塑造生态系统。

## 技术栈
- 前端：React + Taro
- 后端：Node.js
- 数据库：MongoDB
- 消息队列：Redis
- 其他：Python（部分功能）

## 环境要求
- Node.js
- npm 或 yarn
- 微信开发者工具
- Docker（仅Python后端开发需要）
